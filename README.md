# 📏 Quantity Measurement App – Frontend

## 📌 Overview

The **Quantity Measurement App (Frontend)** is a web-based application built using **HTML, CSS, and JavaScript**. It provides a clean and interactive user interface for performing unit conversions after user authentication.

This project is part of a full-stack application where the frontend communicates with a backend (Spring Boot) for handling user authentication and data processing.

---

## 🚀 Features

### 🔐 Authentication

* User Signup with required details
* User Login with validation
* Error handling for invalid credentials
* Backend API integration for authentication

### 🎯 Dashboard

* Welcome screen after successful login
* Multiple measurement categories:

  * Length
  * Temperature
  * Volume

### 🔄 Conversion Functionality

* Input-based unit conversion interface
* Real-time value display
* Simple and intuitive UI for conversions

### 🎨 UI/UX

* Clean and structured layout
* Centered login/signup container
* Card-based dashboard design
* Responsive alignment and spacing

---

## 🛠️ Tech Stack

* **HTML5** – Page structure
* **CSS3** – Styling and layout design
* **JavaScript (Vanilla JS)** – Logic and API handling

---

## ⚠️ Important Notes

* Make sure backend is running before using frontend
* Enable CORS in backend if facing connection issues
* Verify API endpoints and port numbers
* Use browser console (Inspect → Console) for debugging

---

This project demonstrates:
